# 🔐 Password Based Door Lock System using Arduino

## 📖 Project Description
This project implements a password-based door lock system using Arduino. The user enters a password through a keypad, and access is granted only when the correct password is entered.

## 🛠 Components Used
- Arduino UNO
- Keypad
- Servo Motor
- Breadboard
- Jumper Wires
- LCD Display (if used)

## ⚙️ Working Principle
- User enters a password using the keypad.
- Arduino verifies the entered password.
- If the password is correct, the door unlocks.
- If the password is incorrect, access is denied.
